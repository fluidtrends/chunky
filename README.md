## Getting Started

*Prerequisites: *
- *[Node 8](https://nodejs.org/dist/latest-v8.x/) (Node 10 not supported yet)*

**STEP 1:** Install the Chunky Command Line interface

```
npm i -g chunky-cli
```

**STEP 2:** Create a new Chunky Product

```
chunky init
```

**STEP 3:** See your new product in action

```
chunky start web
```
