<hr/>

## Getting Started

**STEP 1:** Install the Chunky CLI

```
npm i -g chunky-cli
```

**STEP 2:** Create a new Chunky Product

```
chunky init MyProduct
```

**STEP 3:** See your new product in action

```
chunky start web
```
